# Python 数据分析学习之旅 (Python Data Analysis Journey)

欢迎来到我的 Python 数据分析学习仓库！

这里记录了我使用 Python 数据分析生态（NumPy, Pandas, Matplotlib, Scikit-learn 等）进行学习、练习和项目实战的完整过程。

## 学习路线图

这个仓库将按照以下模块进行组织和更新：

-   [x] **模块一：NumPy - 数值计算基础**
    -   核心数据结构 `ndarray` 的创建、属性和操作。
    -   向量化运算与广播机制。
    -   常用数学与统计函数。
    -   文件 I/O 与综合练习。
    -   ➡️ **[查看学习笔记](./notebooks/01_NumPy_Core_Concepts.ipynb)**

-   [ ] **模块二：Pandas - 数据处理与分析利器**
    -   `Series` 和 `DataFrame` 的使用。
    -   数据的读取、清洗、转换和筛选。
    -   分组聚合与透视表。
    -   时间序列数据处理。

-   [ ] **模块三：Matplotlib & Seaborn - 数据可视化**
    -   常用图表（折线图、柱状图、散点图等）的绘制。
    -   图表样式与自定义。
    -   使用 Seaborn 进行更美观的统计图表绘制。

-   [ ] **模块四：综合项目实战**
    -   (待定) 真实数据集的探索性数据分析 (EDA)。
    -   (待定) ...

## 如何使用

1.  克隆本仓库到本地：
    ```bash
    git clone https://github.com/YourUsername/Python-Data-Analysis-Journey.git
    ```
2.  进入仓库目录并安装所需依赖（推荐使用虚拟环境）：
    ```bash
    cd Python-Data-Analysis-Journey
    pip install -r requirements.txt 
    ```
    *(注: `requirements.txt` 文件将在后续添加)*

3.  使用 Jupyter Lab 或 VS Code 打开 `notebooks` 文件夹下的 `.ipynb` 文件进行学习。
